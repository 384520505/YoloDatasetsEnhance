# YoloDatasetsEnhance
### yolo数据增强、yolo已标注数据集增强、.txt格式数据集增强；包含旋转、平移、翻转、裁剪、调整亮度和增加噪声6中增强方式随机组合，每张图片扩增到5张。

### 运行环境
- 系统：MAC
- Conda虚拟环境软件包配置：
- Package             Version
- ------------------- ---------
- imgaug              0.4.0
- numpy               1.24.3
- opencv-python       4.1.2.30

** 注意：opencv-python版本不能太高，否则报错 **
